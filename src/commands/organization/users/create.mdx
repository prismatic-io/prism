:::note Organization Roles
See the [users](../configure-prismatic/organization-users.mdx) page for information on roles that users can assume
:::

```bash
# Create an organization user for Susan and grant her the 'Integrator' role
prism organization:users:create \
    --email 'foo@email.com' \
    --name 'Susan Foo' \
    --role $(prism organization:users:roles \
                --columns id \
                --no-header \
                --filter 'name=^Integrator$')
```
